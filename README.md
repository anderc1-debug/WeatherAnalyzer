# 🌤 Weather Data Analyzer

A JavaFX desktop application for analyzing weather data from CSV files, built using **modern Java features (Java 15–23)**.



## Features

- 📂 Load any weather CSV file at runtime
- 📊 Monthly average temperature bar chart
- 🌡️ Days above a custom temperature threshold
- 🌧️ Rainy day count
- 🏷️ Weather category classification (Freezing / Cold / Cool / Warm / Hot)
